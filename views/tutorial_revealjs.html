FRAGMENTOS

Compostos de INDEX e CLASS

INDEX: Indice do fragmento (qual fragmento é exibido na ordem)

CLASS: Tipo de transição do fragmento (como o fragmento se revela na apresentação)

FRAGMENTOS: INDEX

data-fragment-index="#"

# é um numero qualquer (começando de 0)

FRAGMENTOS: CLASS

class="fragment [transição]"

[transição] determina o tipo de transição.

Pode ser omitido.

Nesse caso, usa a transição padrão de fade-in.

FRAGMENTOS: TIPOS DE TRANSIÇÃO

current-visible

grow

shrink

fade-out

highlight red

highlight blue

highlight green

PLANO DE FUNDO

Configure plano de fundo dos slides individualmente usando data-background=""

Você pode configurar qualquer coisa como plano de fundo...

COMO CORES

data-background="#00ffff"

IMAGENS

section data-background="https://static.significados.com.br/foto/ed5a-o-grito_sm.png"

IMAGENS LADO A LADO

section data-background="https://static.significados.com.br/foto/ed5a-o-grito_sm.png"

data-background-repeat="repeat"

data-background-size="100px"

INCLUSIVE, GIFS!

TRANSIÇÕES DE PLANO DE FUNDO

Usando data-background-transition, você pode configurar transições de pano de fundo!

LISTANDO COISAS...

Com tags html ou markdown

  • Listas nao ordenadas usam "ul"
  • Listas ordenadas usam "ol"
  • Dentro deles, existem os "li"
  • O que estiver envolto nos "li" corresponde a um item da lista

COMO FUNCIONA ESSE TROÇO?

  1. Crie a tag "ul" ou "ol"
  2. Dentro dela, crie os seus "li"
  3. Crie suas listas adoidado
